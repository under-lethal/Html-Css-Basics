HTML Image Syntax

In HTML, images are defined with the < img > tag.
The < img > tag is empty, it contains attributes only, and does not have a closing tag.
The src attribute specifies the URL (web adress) of the image:

The alt Attribute

The alt attribute provides an alterate text for an image, if the user for some reason cannot viwe it (because of slow connection, an error in the src attribute, or if the user uses a screen reader).
If a browser cannot find an image, it will display the value of the attribute.

Image Size - Width and Height

You can use the style attribute to specify the width and height of an image.
The values are specified in pixels (use px after the value)
Alternatively, you can use width and height attributes. The values may be specified in pixels by default:
example: < img src="html5.gif" alt="HTML5 Icon" width="128" height="128">

HTML Screen Readers

A screen reader is a software program that reads the HTML code, converts the text, and allows the user to "listen" to the content. Screen readers are useful for people who are blind, visuallt impaired, or learning disabled.

Width and Height, or Style?

Both the width, height, and style attributes are valid in HTML5.
However, we suggest using the style attribute. It prevents internal or external styles sheets from changing the original size of images.

Image Maps

Use the <map> tag to define an image-map. An image-map is an image with clickable areas.
The name attribute ps the <map> tag is associated with the < img >'s usermap attribute and creates a relationship betwen the image and the map.
The <map> tag contains a number of <area> tags, that defines the clickable areas in the image-map
