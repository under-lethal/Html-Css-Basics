XHTML is HTML written as XML.

WHat is XHTML?

  • XHTML stands for EXtensible HyperTert Markup Language
  • XHTML is amost identical to HTML
  • XHTML is stricter than HTML
  • XHTML is HTML defined as an XML application
  • XHTML is supported by all major browsers

Why XHTML?

Many pages on the internet contain "bad" HTML.
This HTML code works fine in the most browsers (even is it does not follow the HTML rules):
Today's market consist of different browsers technologies. Some browsers run on computers, and some browsers run on mobile phones or ohter small devices.
Smaller devices often lack the resources or power to interpret"bad" markup. XML is a markup language where documents must be marked up correctly (be "well-formed")
By combining the strengts of HTML an XML, XHTML was developed.
XHTML is HTML redesigned as XML.

The Most Important Differences from HTML:

Document Structure

  • XHTML DOCTYPE is mandatory
  • The xmlns attribute in <html> is mandatory
  • < html > < head > < title > and < body > are mandatory

XHTML Elements

  • XHTML elements must be properly nested
  • XHTML elements must always be closed
  • XHTML elements must be in lowercase
  • XHTML documents must have One root element

XHTML Attributes

  • Attribute names must be in lower case
  • Attribute values must be quoted
  • Attribute minimizatin is forbidden
