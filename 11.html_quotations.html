HTML Quotation an Citation Elements

HTML for short Quotations
Browsers usually insert quotation marks around the element.

WWF's goal is to: Build a future where people live in harmony with nature.

HTML blockquote for Quotations

Browsers usually indent blockquote elements.
Here is a quote from WWF's website:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

HTML for Abbreviations

The HTML element defines an abbreviation or an acronym.
Making abbreviations can give useful information to browsers, translatio systems and search-engines.

The WHO was founded in 1948.

HTML < address > for a Contact Information

The HTML "< address >" element defines contact information (author/onwer) of a document or an article.
The < address > element is usually diplayed in ITALIC.
Most browsers will add a line break before and after the element.

Example:
Written by John Lee
Visit us at:
Example.com
Box 564, Disneyland
USA

HTML < cite > for Work Tile

The HTML < cite > element defines the title of a work.
Browsers usually display < cite > elements in italic.

Example:

The Scream by Edvard Munch. Painted in 1893.

HTML < bdo > for Bi-Directional Override

The HTML < bdo > element is usd to override the current text direction.

Example:
This text will be written from right to left
