HTML Block an Inline Elements

Every HTML element has a default display value depending on what type of element it is. The default disply value for most elements is block or inline.

Block-level Elements

A block-level element always starts on a new line and takes up the full width available (streches out the left and right as far as it can).
The <div> element is a block-level element.
Examples of block-levels elements:

  • <div>
  • <h1> - <h6>
  • <p>
  • <form>

Inline Elements

An inline element does not start on a new line and only takes up as much width as necessary.

This is an inline < span> element inside a paragraph.
Examples of inline elements:

  • < span >
  • < a >
  • < img >
