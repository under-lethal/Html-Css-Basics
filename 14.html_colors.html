HEX Value

In HTML, a color can also be specified using a hexadecimal value in the form: #RRGGBB, where RR(red), GG(green) and BB (blue) are hexadecimal values betwen 00 an FF (same as decimal0-255).
For example, #F0000 is displayed as red, because red is set to its highest value (FF)and the others are set to the lowest value (00).
